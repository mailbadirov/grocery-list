Grocery List

Чтобы стереть весь список - нажмите на значок ластика слева от поля ввода.
Чтобы вычеркнуть определенную покупку из списка - нажмите по наименованию продукта левой кнопкой мыши.
Чтобы изменить покупку - нажмите по наименованию продукта средней кнопкой мыши. Далее - измените текст и нажмите значок галочки, чтобы сохранить. Для удаления продукта из списка нажмите на крестик.
